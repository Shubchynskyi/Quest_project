<p>Даний додаток створено на <strong>Java/Spring</strong> + <strong>Thymeleaf</strong> і служить для демонстрації навичок роботи з Java та базами даних (PostgreSQL).</p>
<p>Додаток дозволяє користувачам створювати текстові квести із зображеннями, а також проходити їх, використовуючи складний текстовий парсер на Java.</p>
<p>Реалізовано можливість створення користувачів та управління ролями користувачів (адміністратори мають доступ до редагування квестів та інших користувачів, прості користувачі можуть редагувати тільки себе та свої власні квести).</p>
<p>Деплой додатку здійснюється за допомогою CI/CD (GitHub Webhook + Jenkins + Docker).</p>
<p>Код додатку знаходиться за адресою: <a href="https://github.com/Shubchynskyi/Quest_project/tree/master" target="_blank">GitHub Repository</a></p>
<p>Детальний технічний опис доступний у файлі <strong>Readme.MD</strong> репозиторію.</p>
