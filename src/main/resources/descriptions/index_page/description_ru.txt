
<p>Данное приложение создано на <strong>Java/Spring</strong> + <strong>Thymeleaf</strong> и служит для демонстрации навыков работы с Java и базами данных (PostgreSQL).</p>
<p>Приложение позволяет создавать пользователям текстовые квесты с изображениями, а также проходить их, используя сложный текстовый парсер на Java.</p>
<p>Реализована возможность создания пользователей и управление ролями пользователей (администраторы имеют доступ к редактированию квестов и других пользователей, простые пользователи могут редактировать только себя и свои собственные квесты).</p>
<p>Деплой приложения осуществляется посредством CI/CD (GitHub Webhook + Jenkins + Docker).</p>
<p>Код приложения находится по адресу: <a href="https://github.com/Shubchynskyi/Quest_project/tree/master" target="_blank">GitHub Repository</a></p>
<p>Подробное техническое описание доступно в файле <strong>Readme.MD</strong> репозитория.</p>
